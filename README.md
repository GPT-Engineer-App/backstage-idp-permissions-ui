## Setup

```sh
git clone https://github.com/GPT-Engineer-App/backstage-idp-permissions-ui.git
cd backstage-idp-permissions-ui
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
